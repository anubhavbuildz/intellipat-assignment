# Azure Capestone

A concise starter for the Azure capstone project in this workspace.

## Description

This repository contains the code and resources for the Azure capstone assignment. It provides configuration, scripts, and documentation needed to build, deploy, and evaluate the project.

## Prerequisites

- macOS or Linux
- Python 3.8+ (if the project uses Python)
- Node.js 14+ (if the project uses Node)
- Azure CLI installed and logged in (`az login`)

## Setup

1. Clone the repository:

   git clone <repo-url>
   cd azure-capestone

2. Install project dependencies (adjust per project language):

   - Python: `pip install -r requirements.txt`
   - Node: `npm install`

3. Configure Azure credentials and resources using the Azure CLI.

## Running

- See project-specific scripts or README subfolders for run instructions.
- Example (Python):

  python main.py

Example (Node):

  npm start

## Project Structure

- `src/` — source code
- `infra/` — infrastructure or deployment scripts
- `docs/` — documentation
- `tests/` — automated tests
